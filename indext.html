Học về các kí tự đặc biệt trong HTML - HTML Entities

Các kí tự đặc biệt như dấu: lớn hơn, bé hơn, =, $, @ ... trong HTML và các thẻ đóng, mở trong một số trường hợp sẽ không hiển thị trong tài liệu HTML, do vậy chúng ta cần sử dụng HTML Entity.
Để hiểu rõ hơn chúng ta search từ khóa: HTML Entity list. Ví dụ để hiển thị một đoạn code HTML ta làm như sau: 

        <pre>
           <p>Đây là một thẻ p trong HTML được hiển thị khi dùng HTML Entity</p>
           <p> Thẻ pre trong HTML sẽ giúp bạn giữ nguyên định dạng code, 
           bạn viết như thế nào thì sẽ hiển thị như thế ấy</p>
        </pre>

Ngoài ra bạn cũng có thể hiển thị các kí tự đặc biệt khác như: ©, ®, 3², ¼,...
